Glosario
--------

.. glossary::

   ordenar
        Un método que ordena una lista en su lugar, cambiando el contenido de la lista. Eso
        return Ninguno, no es una lista nueva.

    ordenado
        Una función que devuelve una lista ordenada, sin cambiar el original.

    parámetro inverso
        Si es verdadero, la clasificación se realiza en orden inverso.

    parámetro clave
        Si se especifica un valor, debe ser un objeto de función que tome un parámetro.
        Se llamará a la función una vez para cada elemento de la lista que está obteniendo
        ordenado El valor de retorno se utilizará para decorar el artículo con un post-it.
        Nota: Los valores en las notas post-it se utilizan para determinar el orden de clasificación de
        los artículos.
