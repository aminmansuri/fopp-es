Glosario
--------

.. glossary::

    determinista
        Un proceso que es repetible y predecible.

    documentación
        Un lugar donde puede ir para obtener información detallada sobre aspectos de un
        lenguaje de programación.


    módulo
        Un archivo que contiene definiciones y declaraciones de Python destinadas a ser utilizadas en
        otros programas de Python. Los contenidos de un módulo están disponibles para
        el otro programa usando la instrucción *import*  .

    espacio de nombres
        Un sistema de nombres para hacer nombres únicos, para evitar duplicaciones y confusión. Dentro de un espacio de nombres, no hay dos nombres que puedan ser iguales.

    número pseudoaleatorio
        Un número que no es realmente aleatorio, sino que se crea algorítmicamente.

    número aleatorio
        Un número que se genera de tal manera que exhibe aleatoriedad estadística.

    generador de números aleatorios
        Una función que le proporcionará números aleatorios, generalmente entre 0 y 1.


    biblioteca estándar
        Una colección de módulos que forman parte de la instalación normal de Python.
